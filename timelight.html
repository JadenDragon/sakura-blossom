The Timelight Project

What is the Timelight Project

    The Timelight Project was an interactive installation where users move through space, as they trigger various interactive elements that correspond to each era. This could be as simple as by walking in a circle or painting a famous illustration with everyone around. Each interaction is designed to put the user in the time period, making them involved in part of the story themselves. By using a camera to track movement, we created mechanics that involve the users to move and interact with the installation itself. The goal was to blend history with entertaining user participation with the use of new technology.

    This project was an assignment given to us to setup for the Enschede Lights Up Festival & Hengelo LICHT, as a way to promote the respective cities. The Timelight Project was created with the goal of accessibility, so that anyone could enjoy and interact with this installation.
