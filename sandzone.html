About the Project

    Project SandZone was a game prototype that I created during my minor with the goal of learning to make a game in unreal engine. This is a game where you get stuck inside a temple and the only way out is in. The game is an adventure-puzzle game where you unlock the follow-up level by solving puzzles and explore farther inside as you collect treasure and avoide being captured by the golem soldiers.
